工作电脑，稳定使用约1年。目前存在的bug：睡眠唤醒后，显示器无信号输出。
目前在用的两种解决方案：
· 方案一，断开显示器电源，并重新插拔一次HDMI线。
· 方案二，用手机远程连接到桌面，此时显示器自动点亮有画面输出。
